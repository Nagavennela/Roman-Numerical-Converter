# Roman Numeral Converter

A simple web application that converts numbers into Roman numerals.

## Features
- Convert numbers to Roman numerals
- Simple and user-friendly interface
- Built with HTML, CSS, and JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Example
Input: 36  
Output: XXXVI
